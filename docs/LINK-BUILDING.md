# Link Building Plan — First Sale Society

Off-site strategy for earning backlinks once the site is live and indexed.
None of this requires code changes — it's an execution checklist for
whoever runs outreach/growth. Pursue this after Phase 1 (Foundation) launch
is complete and indexed, since backlinks to a not-yet-indexed site waste the
outreach effort.

## 1. Resource Directory Placement

The site's free, un-gated model qualifies it for listing in open-course and
business-resource directories. Submit to:

| Directory | Pitch angle |
|---|---|
| University/business resource hubs (e.g. business school "free resources" pages) | Pitch as a free, modern e-commerce resource suitable for a startup/entrepreneurship curriculum |
| Open online-course aggregators (e.g. course-listing sites) | Submit as an un-gated, operator-led training asset — emphasize "no paywall, no email gate" as the differentiator from most listed dropshipping courses |
| Curated open-source / "awesome list" repositories on GitHub | Submit a pull request adding the ROAS calculator and glossary as resources to relevant marketing/ecommerce awesome-lists |

Before submitting to any specific directory, verify it's still active and
accepting submissions — directory link rot is common, and outreach time is
better spent confirming a target is live than submitting into a dead form.

## 2. Programmatic Community Referrals

Monitor relevant communities for questions this site directly answers, and
respond with genuinely helpful, non-spammy answers that include a relevant
link only when it adds real value to the answer (not as the primary point
of the reply).

- **Reddit**: r/dropshipping, r/ecommerce, r/Entrepreneur — watch for
  questions about ad optimization, supplier transitions, and break-even
  math specifically, since those map directly to this site's strongest
  pages (ROAS calculator, supplier module, glossary).
- **Quora**: search for exact-match questions like "How is break-even ROAS
  calculated?" or "Is CJDropshipping reliable?" and write complete,
  standalone answers — link back only where the linked page genuinely
  extends the answer.
- **Discord/Slack communities**: share the ROAS calculator specifically in
  relevant channels when someone asks a margin/profitability question — it's
  the single highest-utility, most shareable asset on the site.

**Important**: community-platform guidelines on this vary and change. Drive-by
linking that reads as promotional gets removed and can get an account
banned — prioritize answers that would be worth posting even without a link,
then add the link as a bonus, not the goal.

## 3. Podcast Guest Outreach

E-commerce and marketing podcasts are a high-authority backlink channel,
since most include show-notes links to guests' sites/resources. Before
running outreach, verify each target podcast is still actively publishing
(podcast cancellation/rebranding is common) — the list below should be
treated as a researched starting point, not a verified-current list.

Suggested approach: pitch a specific, narrow topic this site has unusually
deep, structured material on (the 6-checkpoint validation framework, the
break-even ROAS calculator, the CJDropshipping → private agent transition)
rather than a generic "have me on to talk about dropshipping" pitch — hosts
get the generic pitch constantly and a specific, useful framing stands out.

Target podcast categories to research and prioritize by authority/relevance:
- Shopify/platform-affiliated shows (highest authority, platform-specific
  angle works well — e.g. ad frameworks, CBO scaling)
- Operator-interview shows focused on margins, logistics, and scaling
  (strong fit for the supplier-transition and break-even content)
- Store-design/CRO-focused shows (fit for the conversion-rate module
  content)
- Beginner-focused entrepreneurship shows (fit for the "free course" angle
  and the homepage's core positioning)

Re-research actual current show names, hosts, and contact info at outreach
time rather than relying on a static list here — podcast rosters and host
changes happen often enough that a hardcoded list would go stale within
months.

## Sequencing

Don't start link building before Phase 1 (Foundation) — sitemap submitted,
core pages indexed — is confirmed in Google Search Console. A backlink to an
unindexed page passes no equity until indexation catches up, and "submit
everywhere on day one" is a worse use of limited outreach time than waiting
3-4 weeks for the foundation to be live.
